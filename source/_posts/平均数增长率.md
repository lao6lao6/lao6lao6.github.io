# 平均数增长率

## 1. 概念定义

平均数增长率是指**平均数的增长率**，即现期平均数相对于基期平均数的增长幅度。

**注意区分：**
- **平均数的增长量** = 现期平均数 − 基期平均数
- **平均数增长率** = (平均数的增长量) / (基期平均数) × 100%

---

## 2. 核心公式

设：
- 总量为 `A`，增长率为 `a`
- 份数为 `B`，增长率为 `b`

则：
- 基期平均数 = \(\dfrac{A}{B}\)
- 现期平均数 = \(\dfrac{A \times (1+a)}{B \times (1+b)}\)

### 平均数增长率公式

\[
\boxed{r = \frac{1+a}{1+b} - 1 = \frac{a - b}{1 + b}}
\]

其中：
- \(r\)：平均数的增长率
- \(a\)：总量的增长率
- \(b\)：份数的增长率

> **记忆技巧**：平均数增长率 = \(\dfrac{\text{分子增长率} - \text{分母增长率}}{1 + \text{分母增长率}}\)

---

## 3. 重要性质

| 情况 | 结论 |
|------|------|
| \(a > b\) | 平均数增长 |
| \(a < b\) | 平均数下降 |
| \(a = b\) | 平均数不变 |
| \(b > -1\) | 公式成立（通常满足） |

---

## 4. 例题讲解

### 例题 1：基础计算

**题目**：  
2023年某公司总产值为1200万元，同比增长20%；员工人数为200人，同比增长10%。求2023年人均产值比2022年增长了百分之几？

**解析**：

- 总量（产值）增长率 \(a = 20\% = 0.2\)
- 份数（人数）增长率 \(b = 10\% = 0.1\)

\[
r = \frac{a - b}{1 + b} = \frac{0.2 - 0.1}{1 + 0.1} = \frac{0.1}{1.1} \approx 0.0909 = 9.09\%
\]

✅ **答案**：9.09%

---

### 例题 2：求分子或分母增长率

**题目**：  
某地区2023年人均收入为5万元，比2022年增长4%；总人口增长2%。求该地区总收入增长率。

**解析**：

已知：人均收入增长率 \(r = 4\% = 0.04\)，人口增长率 \(b = 2\% = 0.02\)

由公式：
\[
r = \frac{a - b}{1 + b} \implies 0.04 = \frac{a - 0.02}{1.02}
\]

\[
a - 0.02 = 0.04 \times 1.02 = 0.0408
\]

\[
a = 0.0608 = 6.08\%
\]

✅ **答案**：总收入增长 6.08%

---

### 例题 3：比较与判断

**题目**：  
某工厂2022年产品总成本增长15%，产品数量增长20%。判断单位产品成本的变化情况。

**解析**：

- \(a = 15\%\)（成本增长率）
- \(b = 20\%\)（数量增长率）

\[
r = \frac{0.15 - 0.20}{1 + 0.20} = \frac{-0.05}{1.2} \approx -0.0417 = -4.17\%
\]

**结论**：单位产品成本**下降**了约 4.17%。

✅ 平均数增长率可以为负，表示下降。

---

## 5. 常见易错点

| 易错点 | 正确理解 |
|--------|----------|
| 直接用 \(a - b\) 作为答案 | 必须除以 \(1 + b\) |
| 混淆平均数的增长量与增长率 | 增长量有单位，增长率是百分数 |
| 忘记 b 可能为负 | 当 b 为负时，分母变小，公式仍适用 |
| 混淆“平均数的增长率”与“总量的增长率” | 两者完全不同 |

---

## 6. 小结

| 要点 | 内容 |
|------|------|
| 核心公式 | \(r = \dfrac{a-b}{1+b}\) |
| 使用场景 | 已知总量、份数各自的增长率，求平均数的增长率 |
| 注意事项 | 分母是 \(1+b\)，不是 \(1+a\) |
| 正负判断 | \(a > b\) 则平均数上升，反之下降 |

---
