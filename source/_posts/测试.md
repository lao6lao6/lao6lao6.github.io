## 2. 文本样式测试

这是普通文本。我们可以使用 **粗体** 来强调重要内容，也可以使用 *斜体* 表示轻微强调，或者使用 ***粗斜体*** 进行强烈强调。

## 5. 代码块测试

### 普通代码块

```python
def fibonacci(n):
    """计算斐波那契数列的第n项"""
    if n <= 0:
        return 0
    elif n == 1:
        return 1
    else:
        return fibonacci(n-1) + fibonacci(n-2)

# 输出前10个斐波那契数
for i in range(10):
    print(f"F({i}) = {fibonacci(i)}")
```
